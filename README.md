# 🚗 404 AutoCare

404 AutoCare is a web application that allows travelers to park their cars at the airport and request car detailing services while they are away. Users can track the service progress and return to a clean and serviced vehicle.

---

## Features

- User Registration and Login
- Phone number collection
- Airport car parking booking
- Service selection
- Booking confirmation page
- Service progress tracking
- Admin management panel
- Booking dashboard for users

---
